# 将数组视为哈希表-原地哈希法

* 题目要求的是常数个辅助空间，而要找的数一定在1--len+1以内
* 因此只需要找1——len，len+1可以在前面没找到的时候自动返回
* 思路：就把 1 这个数放到下标为 0 的位置， 2 这个数放到下标为 1 的位置，按照这种思路整理一遍数组。然后我们再遍历一次数组，第 1 个遇到的它的值不等于下标的那个数，就是我们要找的缺失的第一个正数。

```c
int firstMissingPositive(int* nums, int numsSize) {
    for(int i = 0;i < numsSize;++i){
        while(nums[i]>0&&nums[i]<=numsSize&&nums[nums[i] - 1] != nums[i]){
            int correctIdx = nums[i] - 1;
            int temp = nums[i];
            nums[i] = nums[correctIdx];
            nums[correctIdx] = temp;
        }
    }
    for(int i = 0;i < numsSize;++i){
        if(nums[i] != i +1){
            return i + 1;
        }
    }
    return numsSize + 1;
}
```
