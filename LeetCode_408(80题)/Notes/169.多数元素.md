> Problem: [169. 多数元素](https://leetcode.cn/problems/majority-element/description/)

思路

> 摩尔投票法

解题过程

> 想象一场拔河比赛：

> 候选人是「当前可能的多数元素」（记为 candidate）

> 计数器 count是「候选人的票数优势」

> 规则：同队+1，敌队-1，优势归零就换队长

Code

```c
ßintmajorityElement(int* nums, intnumsSize) {
int candidate = nums[0];
int count = 1;
for(int i = 1 ;i < numsSize;++i){
if(candidate == nums[i]){
           count++;
       }else{
           count--;
if(count == 0){
               candidate = nums[i];
               count = 1;
           }
       }
   }
return candidate;
}
```
