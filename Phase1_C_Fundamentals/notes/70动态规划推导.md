## 爬楼梯（LeetCode 70）

### 题目

```
你爬楼梯，每次上 1 阶或 2 阶。n 阶的楼梯有多少种不同爬法？
n=1 → 1种（1）
n=2 → 2种（1+1, 2）
n=3 → 3种（1+1+1, 1+2, 2+1）
n=4 → 5种
```

### DP 三步法

#### 第一步：定义状态

问自己：**"我想知道什么？"**

想知道：爬到第 n 阶有几种方法。

→ 定义 `dp[i]` = 爬到第 i 阶的方法数

#### 第二步：找递推关系（状态转移方程）

问自己：**"dp[i] 能从哪些已知结果推出来？"**

到达第 i 阶，最后一步有两种可能：

- 从 i-1 跨 1 步上来 → 有 `dp[i-1]` 种方法
- 从 i-2 跨 2 步上来 → 有 `dp[i-2]` 种方法

→ **`dp[i] = dp[i-1] + dp[i-2]`**

这就是整个 DP 的核心：**大状态 = 小状态 + 更小状态**

#### 第三步：确定基础情况（初始化）

递推式需要 i-1 和 i-2，所以 i 至少从 3 开始。i=1 和 i=2 需要手动给出：

```
dp[1] = 1   （只能走1步）
dp[2] = 2   （1+1 或 2）
```
