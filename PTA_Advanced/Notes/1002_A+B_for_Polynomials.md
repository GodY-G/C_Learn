# PAT 1002 — A+B for Polynomials（多项式加法）

https://pintia.cn/problem-sets/994805342720868352/exam/problems/type/7

---

## 题目

两个多项式相加。每个多项式先给项数 K，再给 K 对（指数 N，系数 a）。按指数从大到小输出结果。

```
输入：
2 1 2.4 0 3.2      ← A: 2.4x¹ + 3.2x⁰
2 2 1.5 1 0.5      ← B: 1.5x² + 0.5x¹

输出：
3 2 1.5 1 2.9 0 3.2
↑      ↑    ↑      ↑
3项    x²   x¹     x⁰
系数和=1.5+0=1.5   3.2+0=3.2
           2.4+0.5=2.9
```

---

## 思路

指数范围 0~1000，开一个数组 `coef[1001]`。下标 = 指数，值 = 系数累加和。最后从高到低输出系数 ≠ 0 的项。

---

## C 代码

```c
#include <stdio.h>

int main() {
    double coef[1001] = {0.0};   // coef[i] = 指数 i 的系数和
    int k, exp;
    double coe;

    // 读第一个多项式
    scanf("%d", &k);
    for (int i = 0; i < k; i++) {
        scanf("%d %lf", &exp, &coe);
        coef[exp] += coe;
    }

    // 读第二个多项式
    scanf("%d", &k);
    for (int i = 0; i < k; i++) {
        scanf("%d %lf", &exp, &coe);
        coef[exp] += coe;
    }

    // 统计非零项
    int cnt = 0;
    for (int i = 0; i <= 1000; i++) {
        if (coef[i] != 0.0) cnt++;
    }

    // 输出：指数从高到低
    printf("%d", cnt);
    for (int i = 1000; i >= 0; i--) {
        if (coef[i] != 0.0) {
            printf(" %d %.1f", i, coef[i]);
        }
    }

    return 0;
}
```

---

## 逐行解析

### 1. 数组做哈希

```c
double coef[1001] = {0.0};
```

指数 0~1000 共 1001 个坑位。`coef[3] = 2.5` 表示 `2.5x³`。数组下标直接当指数，O(1) 存取。

### 2. `+=` 而非 `=`

```c
coef[exp] += coe;       // ✅
coef[exp] = coe;        // ❌ 同指数项会被覆盖
```

两个多项式可能有**相同指数**的项（如 x¹ 出现在两个多项式中），用 `+=` 累加。用 `=` 后面的会覆盖前面的。

### 3. 统计非零项

```c
if (coef[i] != 0.0) cnt++;
```

正负系数抵消后可能变成 0（如 `2.5 + (-2.5) = 0`），不算有效项。

### 4. 从大到小输出

```c
for (int i = 1000; i >= 0; i--)   // 不是 0→1000
```

指数越大越优先输出。PAT 要求按指数降序。

### 5. 格式

```c
printf("%d", cnt);                    // 第一项：项数前不加空格
printf(" %d %.1f", i, coef[i]);      // 后面每项：空格+指数+空格+系数(1位小数)
```

输出格式是 `K N₁ a₁ N₂ a₂ ...`，纯空格分隔，没有括号也没有逗号。

---

## 易错点

| 坑 | 说明 |
|------|------|
| `=` 而非 `+=` | 同指数项被覆盖，结果少项 |
| 系数为 0 仍输出 | 抵消后 = 0 的不算一项 |
| `%d` 读浮点数 | 系数是 double，必须 `%lf` 读 `%.1f` 打 |
| 指数顺序反了 | 从 1000→0，不是 0→1000 |
| 所有项抵消 | cnt=0，输出只有 `0`，后面不跟任何东西 |
| `%lf` 用于 double | `float` 用 `%f`，此题系数是 double |

## 手算示例

```
输入：
2 1 2.4 0 3.2      多项式A: 2.4x¹ + 3.2x⁰
2 2 1.5 1 0.5      多项式B: 1.5x² + 0.5x¹

数组：
  coef[0] = 3.2
  coef[1] = 2.4 + 0.5 = 2.9
  coef[2] = 1.5

从高到低遍历：
  i=2: coef[2]=1.5 ≠ 0 → 输出
  i=1: coef[1]=2.9 ≠ 0 → 输出
  i=0: coef[0]=3.2 ≠ 0 → 输出

结果：3 2 1.5 1 2.9 0 3.2 ✓
```
